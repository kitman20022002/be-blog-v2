var

Scope

: Function-scoped or globally-scoped variable.

Hoisting: Variables are hoisted to the top of their function or global scope.

Reassignment: Can be reassigned and redeclared.

{`function example() {
console.log(x); // undefined, due to hoisting
var x = 5;
console.log(x); // 5
}`}

let

Scope: Block-level scoping.

Hoisting: Hoisted to the top of their block but not initialized.

Reassignment: Can be reassigned but not redeclared within the same scope.

{`if (true) {
let y = 10;
console.log(y); // 10
}
console.log(y); // ReferenceError: y is not defined`}

const

Scope: Block-scoped.

Hoisting: Hoisted like let but not initialized.

Reassignment: Cannot be reassigned or redeclared. Must be initialized upon declaration.

{`const z = 30;
z = 50; // TypeError: Assignment to constant variable

const obj = { key: "value" };
obj.key = "newValue"; // Allowed`}
